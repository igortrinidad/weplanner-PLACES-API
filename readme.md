# We Planner Places API

#### Install

- Clone this repository
- Run `composer install`
- Run `cp .env.example .env`
- Run `php artisan key:generate`
- Run `php artisan jwt:secret`
- Set up the database on .env file

#### Endpoint

`weplanner-places-api.dev`

#### Laravel Version

Laravel 5.4
